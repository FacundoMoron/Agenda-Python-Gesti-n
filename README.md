Proyecto final aprobado con **calificación máxima (100/100)** en el curso de *Introducción a la Programación con Python* de la **UTN FRRe**.

## 🎯 Descripción del Proyecto
Este script fue desarrollado para resolver la necesidad de centralizar y gestionar bases de datos de contactos de manera local. Como **Licenciado en Administración**, el enfoque principal fue asegurar la **integridad de los datos** y la **interoperabilidad**, permitiendo que la información generada en Python sea consumida por herramientas tradicionales de oficina y análisis.

## 🚀 Funcionalidades Principales
- **Gestión Integral (CRUD):** Permite crear, leer, actualizar y eliminar registros con validaciones automáticas.
- **Persistencia de Datos:** Almacenamiento permanente en archivos `.txt` para evitar la pérdida de información al cerrar el programa.
- **Validación Lógica:** Filtros para asegurar formatos correctos en teléfonos y correos electrónicos, además de detección de duplicados.
- **Exportación Estratégica:** Generación de archivos `.csv` (delimitados por punto y coma) optimizados para su apertura inmediata en **Microsoft Excel** o **Google Sheets**.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python 3.x
- **Módulos:** `os` (manejo de rutas y archivos del sistema).
- **Formatos de salida:** TXT y CSV.
